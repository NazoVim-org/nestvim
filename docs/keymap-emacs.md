# Emacs キーマップ互換レベル（現時点）

この表は `src/keymap/emacs.rs` の実装に基づく、nestvim の Emacs 互換状況。

## 移動

| キー/操作 | 状態 | メモ |
|---|---|---|
| `C-f` `C-b` `C-n` `C-p` | 対応済み | 文字/行移動。 |
| `C-a` `C-e` | 対応済み | 行頭/行末。 |
| 矢印キー / Home / End / PgUp / PgDn | 対応済み | 非 Emacs 由来キーも利用可能。 |
| `M-f` `M-b` など Meta 系 | 未対応 | Alt/Meta 系移動なし。 |

## 挿入/削除

| キー/操作 | 状態 | メモ |
|---|---|---|
| 文字入力 / Enter / Tab | 対応済み | 直接挿入。 |
| Backspace / Delete | 対応済み | 後退/前方削除。 |
| `C-d` `C-h` | 対応済み | 前方/後退削除。 |
| `C-k` | 対応済み | 行末まで削除（kill）。 |
| `C-w` | 仕様差分あり | 本家は region cut、ここでは `kill_word()`。 |

## 検索

| キー/操作 | 状態 | メモ |
|---|---|---|
| `C-s` / `C-r` (isearch) | 未対応 | インクリメンタル検索なし。 |
| `/` 検索 | 未対応 | Emacs キーマップ内に検索キー未定義。 |

## 保存/終了

| キー/操作 | 状態 | メモ |
|---|---|---|
| `C-x C-s` | 対応済み | 保存。 |
| `C-x C-c` | 対応済み | 終了。 |
| `C-o` | 仕様差分あり | 本家は open-line。ここでは保存トリガー。 |

## Undo/Redo

| キー/操作 | 状態 | メモ |
|---|---|---|
| `C-/` `C-_` `C-?` | 対応済み | Undo 実装あり。 |
| `C-g` | 仕様差分あり | abort 動作のみ。undo quit ではない。 |
| `C-x u` / `C-x C-/` redo 相当 | 未対応 | Redo ルートなし。 |

## 選択・ヤンク

| キー/操作 | 状態 | メモ |
|---|---|---|
| `C-y` | 仕様差分あり | 本家 yank ではなく `yank_pop()` を直接実行。 |
| `M-w` | 未対応 | コピーなし。 |
| `C-space` (mark set) | 未対応 | mark/region 操作なし。 |
| `C-w` / `M-w` の region 連携 | 未対応 | region 概念未実装。 |
