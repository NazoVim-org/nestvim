# Vim キーマップ互換レベル（現時点）

この表は `src/editor.rs` の実装に基づく、nestvim の Vim 互換状況。

## 移動

| キー/操作 | 状態 | メモ |
|---|---|---|
| `h` `j` `k` `l` | 対応済み | 基本カーソル移動。 |
| `w` `b` `e` | 対応済み | 単語移動。 |
| `0` `^` `$` | 未対応 | 専用実装なし。 |
| `gg` / `G` | 仕様差分あり | `G` は対応。`gg` は `g` プレフィックス処理が不完全で本家同等ではない。 |
| `%` | 対応済み | 対応括弧へジャンプ。 |
| `f` `F` `t` `T` `;` `,` | 対応済み | 1文字検索移動と繰り返し。 |

## 挿入/削除

| キー/操作 | 状態 | メモ |
|---|---|---|
| `i` `a` `A` `o` `O` | 対応済み | Insert モード遷移あり。 |
| `R` | 対応済み | Replace モードに入る。 |
| `x` | 対応済み | 1文字削除。 |
| `dd` `dw` | 対応済み | 演算子経由で実装あり。 |
| `cw` `cc` | 対応済み | change 演算子あり。 |
| `D` `C` `S` | 未対応 | 専用実装なし（`s` は実装あり）。 |

## 検索

| キー/操作 | 状態 | メモ |
|---|---|---|
| `/` | 対応済み | コマンドモード検索入力。 |
| `n` `N` | 対応済み | 次/前検索。 |
| `*` | 対応済み | カーソル下単語検索。 |
| `?` | 未対応 | 逆方向検索の直接入力なし。 |

## 保存/終了

| キー/操作 | 状態 | メモ |
|---|---|---|
| `:w` | 対応済み | 保存。 |
| `:q` | 対応済み | 変更あり時は確認あり。 |
| `:wq` `:x` `ZZ` | 仕様差分あり | `:wq` / `:x` は対応。`ZZ` は `z` 系実装が未完成で実質非対応。 |
| `:q!` | 対応済み | 破棄終了。 |

## Undo/Redo

| キー/操作 | 状態 | メモ |
|---|---|---|
| `u` | 対応済み | Undo 実装あり。 |
| `Ctrl-r` | 未対応 | Redo キー割り当てなし。 |
| `.` | 仕様差分あり | repeat はあるが本家の完全互換ではない。 |

## 選択・ヤンク

| キー/操作 | 状態 | メモ |
|---|---|---|
| `v` `V` | 対応済み | Visual character/line。 |
| `y` `yy` `yw` | 対応済み | yank 演算子あり。 |
| `p` `P` | 対応済み | 貼り付け。 |
| テキストオブジェクト `iw` `aw` など | 仕様差分あり | 一部のみ対応。本家の網羅性はない。 |
| Visual Block (`Ctrl-v`) | 未対応 | ブロック選択なし。 |
